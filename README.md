# 💼 Job Portal System

A full-stack **MERN Job Portal application** where users can search and apply for jobs, and recruiters can post and manage job listings.

---

## 🚀 Tech Stack

**Frontend**

* React.js
* Redux Toolkit
* Tailwind CSS
* Vite

**Backend**

* Node.js
* Express.js
* MongoDB
* Mongoose

**Other Tools**

* Axios
* Cloudinary (for file uploads)
* JWT Authentication

---

## ✨ Features

### 👤 User

* User Registration & Login
* Update Profile
* Browse Jobs
* Apply for Jobs
* View Applied Jobs

### 🏢 Admin / Recruiter

* Create Company
* Post Jobs
* Manage Jobs
* View Applicants

## ⚙️ Installation

### Clone Repository

```
git clone https://github.com/sharfealam900/Job-Portal-System.git
```

### Backend Setup

```
cd backend
npm install
npm run dev
```

### Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in the backend folder.

Example:

```
PORT=8000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
